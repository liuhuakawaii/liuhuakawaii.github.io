💡 Shader 进阶 - 光照与材质系统的物理基础 ​

🌟 为什么需要光照? ​

在之前的文章中,我们主要处理二维图案和纹理。但要创造真实的三维世界,必须理解光如何与物体表面相互作用。

观察现实世界:

  • 🏀 篮球的橡胶表面是粗糙的漫反射
  • 🔮 玻璃球有清晰的镜面高光和折射
  • 🪨 岩石粗糙、凹凸不平,光照不均
  • 🥇 金属会强烈反射环境

这些视觉差异的核心就是光照模型!

📐 光照基础 - 从物理到数学 ​

🎯 光照三要素 ​

任何光照计算都基于三个向量:

         N (法线)
          ↑
          │
          │
      ────●──── 表面
         ╱ ╲
    L ╱     ╲ V
     ╱       ╲
    ╱         ╲
  光源        观察者

N = Normal (法线方向)
L = Light Direction (指向光源)
V = View Direction (指向摄像机)

核心概念:

  • 法线 (Normal): 垂直于表面的方向,决定表面朝向
  • 入射光 (Incident Light): 从光源到表面
  • 反射光 (Reflected Light): 从表面到观察者

🌈 光的两种反射 ​

类型物理现象视觉效果材质示例
漫反射 (Diffuse)光均匀散射到各个方向柔和、无光泽纸张、布料、粗糙石头
镜面反射 (Specular)光沿特定方向反射高光、光泽金属、塑料、水面
漫反射:                镜面反射:
   ↗  ↑  ↖              ↑
  ↗   ↑   ↖             │ R (反射光)
 ●─────────●            ●
  ↘   │   ↙            ╱
   ↘  ↓  ↙            ╱ L (入射光)
                      ╱
光向所有方向散射      光沿镜面反射方向

🔆 经典光照模型 ​

1️⃣ Lambert 漫反射模型 (最简单) ​

物理原理: 表面亮度与入射角度的余弦成正比

强度 = max(N · L, 0)

其中:
- N · L = 点积,计算余弦值
- max(..., 0) = 背面不受光照

数学解释:

光垂直照射 (θ=0°):     光斜射 (θ=60°):      光从背面来 (θ=120°):
    L                    L                     ╲ L
    ↓                   ╱                       ╲
    ●──N               ●──N                     ●──N
                      ╱                         ╱
cos(0°) = 1.0       cos(60°) = 0.5          cos(120°) = -0.5 → 0.0
最亮                中等亮度                  不亮 (背光)

🔧 GLSL 实现 ​

glsl
#ifdef GL_ES
precision mediump float;
#endif

uniform vec3 u_lightPos;    // 光源位置 (世界空间)
uniform vec3 u_lightColor;  // 光源颜色
uniform vec3 u_objectColor; // 物体本身颜色

varying vec3 v_normal;      // 从顶点着色器传来的法线
varying vec3 v_position;    // 片元的世界坐标

void main() {
    // 📐 归一化法线
    vec3 N = normalize(v_normal);
    
    // 🌟 计算光照方向 (从片元指向光源)
    vec3 L = normalize(u_lightPos - v_position);
    
    // 🧮 Lambert 余弦定律
    float diff = max(dot(N, L), 0.0);
    
    // 🎨 最终颜色 = 物体颜色 × 光照颜色 × 漫反射强度
    vec3 diffuse = u_objectColor * u_lightColor * diff;
    
    gl_FragColor = vec4(diffuse, 1.0);
}

顶点着色器 (传递法线):

glsl
attribute vec3 position;
attribute vec3 normal;

uniform mat4 modelMatrix;
uniform mat4 viewMatrix;
uniform mat4 projectionMatrix;
uniform mat3 normalMatrix;  // = transpose(inverse(mat3(modelMatrix)))

varying vec3 v_normal;
varying vec3 v_position;

void main() {
    // 🌍 世界空间坐标
    vec4 worldPos = modelMatrix * vec4(position, 1.0);
    v_position = worldPos.xyz;
    
    // 🧭 变换法线到世界空间
    v_normal = normalMatrix * normal;
    
    // 📹 投影到屏幕
    gl_Position = projectionMatrix * viewMatrix * worldPos;
}

2️⃣ Phong 光照模型 (经典三分量) ​

组成: 环境光 + 漫反射 + 镜面反射

最终颜色 = Ambient + Diffuse + Specular

🌍 环境光 (Ambient) ​

模拟间接光照 (天空光、反射光)

glsl
vec3 ambient = u_ambientStrength * u_lightColor * u_objectColor;

☀️ 漫反射 (Diffuse) ​

Lambert 模型 (如上)

✨ 镜面反射 (Specular) ​

原理: 反射方向接近视线方向时产生高光

      N
      ↑
      │
  L ╱   ╲ R (反射方向)
   ╱     ╲
  ╱       ╲ V (视线)
 ╱         ↘
光源        观察者

镜面强度 = (R · V)^shininess

反射向量公式:

R = 2(N · L)N - L

或使用GLSL内置函数: R = reflect(-L, N)

🔧 完整 Phong 实现 ​

glsl
uniform vec3 u_lightPos;
uniform vec3 u_viewPos;       // 摄像机位置
uniform vec3 u_lightColor;
uniform vec3 u_objectColor;

uniform float u_ambientStrength;  // 环境光强度 (0.1)
uniform float u_shininess;        // 高光锐度 (32.0)

varying vec3 v_normal;
varying vec3 v_position;

void main() {
    vec3 N = normalize(v_normal);
    vec3 L = normalize(u_lightPos - v_position);
    vec3 V = normalize(u_viewPos - v_position);
    
    // 🌍 1. 环境光
    vec3 ambient = u_ambientStrength * u_lightColor;
    
    // ☀️ 2. 漫反射
    float diff = max(dot(N, L), 0.0);
    vec3 diffuse = diff * u_lightColor;
    
    // ✨ 3. 镜面反射 (Phong模型)
    vec3 R = reflect(-L, N);  // 计算反射方向
    float spec = pow(max(dot(R, V), 0.0), u_shininess);
    vec3 specular = spec * u_lightColor;
    
    // 🎨 合成最终颜色
    vec3 result = (ambient + diffuse + specular) * u_objectColor;
    
    gl_FragColor = vec4(result, 1.0);
}

参数效果:

shininess高光效果材质示例
2⭕ 大而模糊粗糙塑料
32🔘 中等大小光滑塑料
128⚪ 小而锐利抛光金属

3️⃣ Blinn-Phong 模型 (性能优化版) ​

问题: Phong 模型在某些角度下会产生不真实的高光消失

改进: 使用半程向量 (Half Vector) 代替反射向量

      N
      ↑ H (半程向量)
      │╱
      ╱
  L ╱   ╲ V
   ╱     ╲
  ╱       ╲

H = normalize(L + V)

镜面强度 = (N · H)^shininess

优势:

  • ✅ 计算更简单 (无需 reflect())
  • ✅ 高光更自然
  • ✅ 业界标准 (Three.js MeshPhongMaterial 使用此模型)

🔧 实现 ​

glsl
void main() {
    vec3 N = normalize(v_normal);
    vec3 L = normalize(u_lightPos - v_position);
    vec3 V = normalize(u_viewPos - v_position);
    
    // 🌟 计算半程向量
    vec3 H = normalize(L + V);
    
    // 环境光 + 漫反射 (同 Phong)
    vec3 ambient = u_ambientStrength * u_lightColor;
    float diff = max(dot(N, L), 0.0);
    vec3 diffuse = diff * u_lightColor;
    
    // ✨ Blinn-Phong 镜面反射
    float spec = pow(max(dot(N, H), 0.0), u_shininess);
    vec3 specular = spec * u_lightColor;
    
    vec3 result = (ambient + diffuse + specular) * u_objectColor;
    
    gl_FragColor = vec4(result, 1.0);
}

🗺️ 法线贴图 (Normal Mapping) - 细节的魔法 ​

❓ 为什么需要法线贴图? ​

问题: 3D模型的面数有限,无法表现所有细节

低模 (1000面):         法线贴图后:
  ╱╲╱╲╱╲               ∿∾∿∾∿∾
 ╱  ╲  ╱              ∾∿∾∿∾∿∾
╱────╲╱               ∿∾∿∾∿∾∿

平滑但缺细节           看起来凹凸不平!
                      (实际几何没变)

解决方案: 用贴图存储每个像素的法线方向

📐 切线空间 (Tangent Space) ​

法线贴图存储的是相对于表面的法线,需要一个局部坐标系:

        N (法线)
        ↑
        │
    ────●──── 表面
       ╱│
      ╱ │
     T  B
     
T = Tangent (切线,沿U方向)
B = Bitangent (副切线,沿V方向)
N = Normal (法线)

TBN 矩阵: 将切线空间转到世界空间

世界法线 = TBN × 贴图法线

🔧 实现步骤 ​

步骤1: 顶点着色器计算 TBN ​

glsl
attribute vec3 position;
attribute vec3 normal;
attribute vec2 uv;
attribute vec3 tangent;  // 切线 (需要预计算或生成)

uniform mat4 modelMatrix;
uniform mat4 viewMatrix;
uniform mat4 projectionMatrix;
uniform mat3 normalMatrix;

varying vec2 v_uv;
varying vec3 v_position;
varying mat3 v_TBN;  // TBN矩阵

void main() {
    vec4 worldPos = modelMatrix * vec4(position, 1.0);
    v_position = worldPos.xyz;
    v_uv = uv;
    
    // 🧭 构建 TBN 矩阵
    vec3 T = normalize(normalMatrix * tangent);
    vec3 N = normalize(normalMatrix * normal);
    
    // Gram-Schmidt 正交化 (确保T垂直于N)
    T = normalize(T - dot(T, N) * N);
    
    // 计算副切线
    vec3 B = cross(N, T);
    
    // 组装 TBN 矩阵
    v_TBN = mat3(T, B, N);
    
    gl_Position = projectionMatrix * viewMatrix * worldPos;
}

步骤2: 片元着色器采样法线贴图 ​

glsl
uniform sampler2D u_normalMap;
uniform sampler2D u_diffuseMap;
uniform vec3 u_lightPos;
uniform vec3 u_viewPos;

varying vec2 v_uv;
varying vec3 v_position;
varying mat3 v_TBN;

void main() {
    // 🗺️ 从法线贴图采样
    vec3 tangentNormal = texture2D(u_normalMap, v_uv).rgb;
    
    // 🔄 从 [0,1] 映射到 [-1,1]
    tangentNormal = tangentNormal * 2.0 - 1.0;
    
    // 🌍 转换到世界空间
    vec3 N = normalize(v_TBN * tangentNormal);
    
    // 其余部分同 Blinn-Phong
    vec3 L = normalize(u_lightPos - v_position);
    vec3 V = normalize(u_viewPos - v_position);
    vec3 H = normalize(L + V);
    
    float diff = max(dot(N, L), 0.0);
    float spec = pow(max(dot(N, H), 0.0), 32.0);
    
    // 🎨 基础颜色从 diffuse 贴图采样
    vec3 baseColor = texture2D(u_diffuseMap, v_uv).rgb;
    
    vec3 ambient = 0.1 * baseColor;
    vec3 diffuse = diff * baseColor;
    vec3 specular = spec * vec3(1.0);
    
    vec3 color = ambient + diffuse + specular;
    
    gl_FragColor = vec4(color, 1.0);
}

🎨 法线贴图的颜色编码 ​

法线贴图看起来是蓝紫色的:

颜色含义:
- R通道 (红): X轴分量 (水平)
- G通道 (绿): Y轴分量 (垂直)
- B通道 (蓝): Z轴分量 (朝向观察者)

平坦表面: RGB(128, 128, 255) 
         = 法线(0, 0, 1) 
         = 垂直朝上

🎯 强度控制 ​

调整法线贴图的影响强度:

glsl
// 采样法线
vec3 tangentNormal = texture2D(u_normalMap, v_uv).rgb;
tangentNormal = tangentNormal * 2.0 - 1.0;

// 🎛️ 控制强度 (0 = 完全平滑, 1 = 完整效果)
float normalStrength = 0.5;
tangentNormal.xy *= normalStrength;

// 重新归一化
tangentNormal = normalize(tangentNormal);

vec3 N = normalize(v_TBN * tangentNormal);

🏔️ 视差贴图 (Parallax Mapping) - 假的凹凸 ​

🎯 原理 ​

法线贴图只改变光照,不改变几何。视差贴图进一步伪造深度感:

侧视图:

    观察者
      │V
      │  ╲
      │   ╲  实际采样点
      │    ●───────┐
      │   ╱        │ 偏移
  ────●───────────┘
   原始UV      高度贴图

根据视角和高度,偏移UV坐标

🔧 基础视差贴图 ​

glsl
uniform sampler2D u_diffuseMap;
uniform sampler2D u_heightMap;  // 高度图 (灰度)
uniform float u_heightScale;    // 高度缩放 (0.01 ~ 0.1)

varying vec2 v_uv;
varying vec3 v_viewDirTangent;  // 视线方向 (切线空间)

void main() {
    // 🗺️ 采样高度
    float height = texture2D(u_heightMap, v_uv).r;
    
    // 📐 计算UV偏移
    vec2 offset = v_viewDirTangent.xy * (height * u_heightScale);
    
    // 🎯 偏移后的UV
    vec2 parallaxUV = v_uv - offset;
    
    // 🎨 用偏移后的UV采样其他贴图
    vec3 color = texture2D(u_diffuseMap, parallaxUV).rgb;
    
    gl_FragColor = vec4(color, 1.0);
}

顶点着色器补充:

glsl
varying vec3 v_viewDirTangent;

void main() {
    // ... (之前的代码)
    
    // 🌍 世界空间的视线方向
    vec3 viewDir = v_position - u_viewPos;
    
    // 🔄 转到切线空间
    v_viewDirTangent = viewDir * v_TBN;  // 注意:乘以TBN的转置
}

🚀 陡峭视差贴图 (Steep Parallax) ​

基础视差在陡峭角度会失真,改进版使用分层采样:

glsl
uniform sampler2D u_heightMap;
uniform float u_heightScale;

const float minLayers = 8.0;
const float maxLayers = 32.0;

vec2 parallaxMapping(vec2 uv, vec3 viewDir) {
    // 📊 根据视角动态调整层数
    float numLayers = mix(maxLayers, minLayers, abs(dot(vec3(0.0, 0.0, 1.0), viewDir)));
    
    // 📏 每层的深度步长
    float layerDepth = 1.0 / numLayers;
    float currentLayerDepth = 0.0;
    
    // 📐 每层的UV偏移
    vec2 deltaUV = viewDir.xy * u_heightScale / numLayers;
    
    vec2 currentUV = uv;
    float currentHeight = texture2D(u_heightMap, currentUV).r;
    
    // 🔄 逐层下降,直到找到交点
    while(currentLayerDepth < currentHeight) {
        currentUV -= deltaUV;
        currentHeight = texture2D(u_heightMap, currentUV).r;
        currentLayerDepth += layerDepth;
    }
    
    return currentUV;
}

void main() {
    vec2 parallaxUV = parallaxMapping(v_uv, normalize(v_viewDirTangent));
    
    vec3 color = texture2D(u_diffuseMap, parallaxUV).rgb;
    gl_FragColor = vec4(color, 1.0);
}

🎯 视差遮蔽贴图 (POM) ​

最高质量,添加插值优化:

glsl
vec2 parallaxOcclusionMapping(vec2 uv, vec3 viewDir) {
    // ... (分层下降,同上)
    
    // 🎯 找到交点后,进行线性插值优化
    vec2 prevUV = currentUV + deltaUV;
    
    float afterDepth = currentHeight - currentLayerDepth;
    float beforeDepth = texture2D(u_heightMap, prevUV).r - currentLayerDepth + layerDepth;
    
    float weight = afterDepth / (afterDepth - beforeDepth);
    vec2 finalUV = mix(currentUV, prevUV, weight);
    
    return finalUV;
}

🌈 PBR 基础 - 基于物理的渲染 ​

🎯 什么是 PBR? ​

Physically Based Rendering (基于物理的渲染) 是现代游戏引擎的标准:

  • ✅ 基于真实物理
  • ✅ 参数直观 (粗糙度、金属度)
  • ✅ 在任何光照下都正确

📊 核心参数 ​

参数含义范围示例
Albedo基础颜色 (不含光照)RGB木头: (0.6, 0.4, 0.2)
Metallic金属度0~1金属: 1.0, 非金属: 0.0
Roughness粗糙度0~1镜面: 0.0, 粗糙: 1.0
AO环境遮蔽0~1缝隙: 0.0, 开阔: 1.0

🔧 简化 PBR 实现 ​

glsl
uniform sampler2D u_albedoMap;
uniform sampler2D u_metallicMap;
uniform sampler2D u_roughnessMap;
uniform sampler2D u_normalMap;
uniform sampler2D u_aoMap;

uniform vec3 u_lightPositions[4];  // 多光源
uniform vec3 u_lightColors[4];
uniform vec3 u_camPos;

varying vec2 v_uv;
varying vec3 v_position;
varying mat3 v_TBN;

const float PI = 3.14159265359;

// 🎨 Fresnel方程 (Schlick近似)
vec3 fresnelSchlick(float cosTheta, vec3 F0) {
    return F0 + (1.0 - F0) * pow(1.0 - cosTheta, 5.0);
}

// 📊 法线分布函数 (GGX/Trowbridge-Reitz)
float distributionGGX(vec3 N, vec3 H, float roughness) {
    float a = roughness * roughness;
    float a2 = a * a;
    float NdotH = max(dot(N, H), 0.0);
    float NdotH2 = NdotH * NdotH;
    
    float nom = a2;
    float denom = (NdotH2 * (a2 - 1.0) + 1.0);
    denom = PI * denom * denom;
    
    return nom / denom;
}

// 📐 几何遮蔽函数
float geometrySchlickGGX(float NdotV, float roughness) {
    float r = (roughness + 1.0);
    float k = (r * r) / 8.0;
    
    float nom = NdotV;
    float denom = NdotV * (1.0 - k) + k;
    
    return nom / denom;
}

float geometrySmith(vec3 N, vec3 V, vec3 L, float roughness) {
    float NdotV = max(dot(N, V), 0.0);
    float NdotL = max(dot(N, L), 0.0);
    float ggx2 = geometrySchlickGGX(NdotV, roughness);
    float ggx1 = geometrySchlickGGX(NdotL, roughness);
    
    return ggx1 * ggx2;
}

void main() {
    // 🗺️ 采样材质贴图
    vec3 albedo = texture2D(u_albedoMap, v_uv).rgb;
    float metallic = texture2D(u_metallicMap, v_uv).r;
    float roughness = texture2D(u_roughnessMap, v_uv).r;
    float ao = texture2D(u_aoMap, v_uv).r;
    
    // 🧭 法线
    vec3 tangentNormal = texture2D(u_normalMap, v_uv).rgb * 2.0 - 1.0;
    vec3 N = normalize(v_TBN * tangentNormal);
    vec3 V = normalize(u_camPos - v_position);
    
    // ✨ 计算 F0 (垂直入射时的菲涅尔反射率)
    vec3 F0 = vec3(0.04);  // 非金属默认值
    F0 = mix(F0, albedo, metallic);  // 金属使用albedo作为F0
    
    // 💡 遍历所有光源
    vec3 Lo = vec3(0.0);
    for(int i = 0; i < 4; i++) {
        vec3 L = normalize(u_lightPositions[i] - v_position);
        vec3 H = normalize(V + L);
        
        float distance = length(u_lightPositions[i] - v_position);
        float attenuation = 1.0 / (distance * distance);  // 平方衰减
        vec3 radiance = u_lightColors[i] * attenuation;
        
        // 🧮 Cook-Torrance BRDF
        float NDF = distributionGGX(N, H, roughness);
        float G = geometrySmith(N, V, L, roughness);
        vec3 F = fresnelSchlick(max(dot(H, V), 0.0), F0);
        
        vec3 numerator = NDF * G * F;
        float denominator = 4.0 * max(dot(N, V), 0.0) * max(dot(N, L), 0.0) + 0.0001;
        vec3 specular = numerator / denominator;
        
        // 🎨 能量守恒: kS + kD = 1
        vec3 kS = F;  // 镜面反射比例
        vec3 kD = vec3(1.0) - kS;
        kD *= 1.0 - metallic;  // 金属没有漫反射
        
        float NdotL = max(dot(N, L), 0.0);
        Lo += (kD * albedo / PI + specular) * radiance * NdotL;
    }
    
    // 🌍 环境光 (简化版)
    vec3 ambient = vec3(0.03) * albedo * ao;
    
    vec3 color = ambient + Lo;
    
    // 🎨 HDR色调映射 (Reinhard)
    color = color / (color + vec3(1.0));
    
    // 📸 Gamma校正
    color = pow(color, vec3(1.0 / 2.2));
    
    gl_FragColor = vec4(color, 1.0);
}
