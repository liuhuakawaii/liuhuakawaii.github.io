Shader 进阶 - 分形与程序化图案的奥秘 ​

1. 什么是分形？为什么它如此重要? ​

在上一篇文章中,我们学习了如何生成单一层次的噪声。但观察自然界,你会发现一个惊人的规律:

自相似性 (Self-Similarity) - 无论你放大多少倍,结构看起来都是相似的!

看看这些例子:

自然现象分形特征
🌳 树木树枝的结构和整棵树的形状相似
⛰️ 山脉小山丘和大山脉的轮廓形状一致
☁️ 云朵云的边缘在任何尺度都是复杂的
🌊 海岸线无论多近看,海岸线都充满细节
❄️ 雪花六边形的每个分支都有相似的结构

这就是分形 (Fractal) 的本质 - 在不同尺度上重复相似的模式。

1.1 分形在 Shader 中的价值 ​

问题: 单层噪声看起来太"光滑"、太"人工"

单层 Perlin Noise:      多层分形噪声:
   ～～～～～～              ～∿～∾～∿～∾～
  ～～～～～～～            ∿～∾～∿～∾～∿～
 ～～～～～～～～           ～∾～∿～∾～∿～∾
   平滑但单调              细节丰富、自然

解决方案: 叠加多个频率的噪声,就像自然界一样!

2. 分形布朗运动 (Fractal Brownian Motion, fBm) ​

2.1 数学原理 ​

fBm 的核心思想: 将不同频率和振幅的噪声叠加起来

数学公式:

fBm(x) = Σ(i=0 to octaves) amplitude(i) × noise(frequency(i) × x)

其中:
- frequency(i) = initial_frequency × lacunarity^i
- amplitude(i) = initial_amplitude × gain^i

参数解析:

参数含义典型值效果
octaves叠加层数4-8越多越细致,但性能下降
lacunarity频率倍增系数2.0每层频率的增长速度
gain振幅衰减系数0.5每层振幅的衰减速度
initial frequency初始频率1.0基础波长
initial amplitude初始振幅0.5基础强度

可视化理解 ​

想象你在画一座山:

第1层 (octave 0):  基础轮廓
amplitude = 0.5    ╱‾‾‾╲___
frequency = 1.0   ╱        ╲___

第2层 (octave 1):  添加中等细节
amplitude = 0.25   ╱‾╲_╱‾╲_
frequency = 2.0   ╱  ╲╱  ╲╱

第3层 (octave 2):  添加小细节
amplitude = 0.125  ╱‾╲╱‾╲╱‾╲╱
frequency = 4.0   ╱ ╲╱ ╲╱ ╲╱

最终叠加:         ╱‾╲_╱‾╲_
                ╱‾╲ ╲╱ ╲╱╲_
               ╱  ╲╲╱╲_╱  ╲╱╲_
                  细节丰富的山脉!

关键规律:

  • 🔊 低频 (大波长): 控制整体形状
  • 🔉 中频: 添加中等细节
  • 🔈 高频 (小波长): 添加微小细节

🔧 基础 fBm 实现 ​

glsl
#ifdef GL_ES
precision mediump float;
#endif

uniform vec2 u_resolution;
uniform float u_time;

// 📐 2D随机函数 (从上一篇复制)
float random(vec2 st) {
    return fract(sin(dot(st, vec2(12.9898, 78.233))) * 43758.5453123);
}

// 🌊 2D Value Noise (从上一篇复制)
float noise(vec2 st) {
    vec2 i = floor(st);
    vec2 f = fract(st);
    
    float a = random(i);
    float b = random(i + vec2(1.0, 0.0));
    float c = random(i + vec2(0.0, 1.0));
    float d = random(i + vec2(1.0, 1.0));
    
    vec2 u = f * f * (3.0 - 2.0 * f);
    
    return mix(mix(a, b, u.x), mix(c, d, u.x), u.y);
}

// 🏔️ 分形布朗运动 (核心!)
float fbm(vec2 st) {
    float value = 0.0;     // 累积值
    float amplitude = 0.5; // 初始振幅
    float frequency = 1.0; // 初始频率
    
    // ⚙️ 配置参数
    const int octaves = 6;      // 叠加6层
    const float lacunarity = 2.0; // 频率每层翻倍
    const float gain = 0.5;      // 振幅每层减半
    
    // 🔄 叠加多层噪声
    for(int i = 0; i < octaves; i++) {
        value += amplitude * noise(st * frequency);
        
        frequency *= lacunarity;  // 频率增加
        amplitude *= gain;        // 振幅衰减
    }
    
    return value;
}

void main() {
    vec2 st = gl_FragCoord.xy / u_resolution.xy;
    st *= 3.0;  // 缩放坐标
    
    float n = fbm(st);
    
    vec3 color = vec3(n);
    gl_FragColor = vec4(color, 1.0);
}

参数变化视觉效果应用场景

octaves ↓更光滑,性能提升实时应用,移动端

gain ↑细节层影响更大尖锐纹理(闪电)
gain ↓基础层主导平滑纹理(沙丘)

🌀 域扭曲 (Domain Warping) - 噪声的噪声 ​

🎯 核心思想 ​

不直接使用噪声,而是用噪声来扭曲坐标系,然后在扭曲的空间中采样噪声!

传统方式:              域扭曲:
noise(x, y)           noise(x + noise(x,y), y + noise(x,y))
                            ↑
                      用噪声偏移坐标!

可视化:

原始网格:            扭曲后的网格:
┌─┬─┬─┬─┐           ╱╲╱╲╱╲
├─┼─┼─┼─┤          ╱╲ ╲╱ ╱╲
├─┼─┼─┼─┤         ╱  ╲╱╲╱  ╲
└─┴─┴─┴─┘        ╱   ╱╲╱╲   ╲
规整              扭曲变形

3.2 基础实现 ​

glsl
float noise(vec2 st) {
    // ... (之前的噪声函数)
}

float fbm(vec2 st) {
    // ... (之前的 fBm 函数)
}

// 🌀 域扭曲版本的 fBm
float fbmWarped(vec2 st) {
    // 1️⃣ 第一次扭曲
    vec2 q = vec2(
        fbm(st + vec2(0.0, 0.0)),
        fbm(st + vec2(5.2, 1.3))
    );
    
    // 2️⃣ 第二次扭曲 (用第一次的结果)
    vec2 r = vec2(
        fbm(st + 4.0 * q + vec2(1.7, 9.2)),
        fbm(st + 4.0 * q + vec2(8.3, 2.8))
    );
    
    // 3️⃣ 最终采样
    return fbm(st + 4.0 * r);
}

void main() {
    vec2 st = gl_FragCoord.xy / u_resolution.xy;
    st *= 3.0;
    
    float n = fbmWarped(st);
    
    vec3 color = vec3(n);
    gl_FragColor = vec4(color, 1.0);
}

效果: 产生类似火焰、液体、有机生物的复杂纹理!

void main() {

📐 分形几何 - 数学之美 ​

除了噪声,Shader 还可以绘制经典的分形几何图形!

🔺 谢尔宾斯基三角形 (Sierpiński Triangle) ​

规则: 从一个三角形开始,递归地移除中间的三角形

迭代0:     迭代1:      迭代2:
  ▲         ▲          ▲
 ▼▼        ▲ ▲        ▲ ▲
▼▼▼▼      ▲▼▼▲      ▲ ▲ ▲ ▲
                    ▲▼▼▲▼▼▲

🔧 实现方法 ​

glsl
uniform vec2 u_resolution;

float triangle(vec2 st) {
    // 使用 SDF 定义三角形
    float d = abs(st.x) * 0.866025 + st.y * 0.5;
    return step(d, 0.5) * step(-st.y, 0.5);
}

float sierpinski(vec2 st, int iterations) {
    float result = triangle(st);
    
    vec2 scale = vec2(1.0);
    vec2 offset = vec2(0.0);
    
    for(int i = 0; i < 5; i++) {
        if(i >= iterations) break;
        
        scale *= 0.5;
        
        // 检查是否在中间的三角形内 (要移除的部分)
        vec2 st_scaled = (st - vec2(0.0, -0.25)) / 0.5;
        if(triangle(st_scaled) > 0.5) {
            result = 0.0;
            break;
        }
        
        // 递归到子三角形
        st = st * 2.0;
        if(st.y < 0.0) {
            st.y += 1.0;
        } else if(st.x < 0.0) {
            st += vec2(0.5, -0.5);
        } else {
            st += vec2(-0.5, -0.5);
        }
    }
    
    return result;
}

void main() {
    vec2 st = gl_FragCoord.xy / u_resolution.xy;
    st = st * 2.0 - 1.0;
    st.x *= u_resolution.x / u_resolution.y;
    
    float pattern = sierpinski(st, 5);
    
    vec3 color = vec3(pattern);
    gl_FragColor = vec4(color, 1.0);
}

🌀 曼德布洛特集 (Mandelbrot Set) ​

数学定义:

对于复数 c,迭代公式:
z₀ = 0
z_{n+1} = z_n² + c

如果 |z_n| 在迭代中保持有界 (不趋于无穷),则 c 属于曼德布洛特集

🔧 实现 ​

glsl
uniform vec2 u_resolution;
uniform vec2 u_mouse;

void main() {
    vec2 st = gl_FragCoord.xy / u_resolution.xy;
    st = st * 2.0 - 1.0;
    st.x *= u_resolution.x / u_resolution.y;
    
    // 🎯 缩放到感兴趣的区域
    vec2 c = st * 2.0 - vec2(0.5, 0.0);
    
    // 🌀 迭代计算
    vec2 z = vec2(0.0);
    int maxIterations = 100;
    int iterations = 0;
    
    for(int i = 0; i < 100; i++) {
        // z² + c (复数乘法展开)
        z = vec2(
            z.x * z.x - z.y * z.y,  // 实部
            2.0 * z.x * z.y         // 虚部
        ) + c;
        
        // 检查是否发散
        if(length(z) > 2.0) {
            iterations = i;
            break;
        }
    }
    
    // 🎨 根据迭代次数着色
    float brightness = float(iterations) / float(maxIterations);
    
    // 彩虹色映射
    vec3 color = 0.5 + 0.5 * cos(
        6.28318 * (brightness + vec3(0.0, 0.33, 0.67))
    );
    
    gl_FragColor = vec4(color, 1.0);
}

效果: 无限复杂的分形边界,可以无限放大!

🌿 科赫雪花 (Koch Snowflake) ​

规则: 将每条直线段的中间三分之一替换为等边三角形的两边

迭代0: _______________
迭代1: ____╱╲____
迭代2: __╱╲╱  ╲╱╲__

这个需要用 SDF 和布尔运算实现,较为复杂,建议使用递归绘制。

🎨 程序化图案设计 ​

📏 瓷砖图案 (Tiling Patterns) ​

利用 mod 和 fract 创建重复图案:

glsl
void main() {
    vec2 st = gl_FragCoord.xy / u_resolution.xy;
    st *= 10.0;  // 10x10 网格
    
    // 🎨 方案1: 棋盘格
    vec2 grid = floor(st);
    float checker = mod(grid.x + grid.y, 2.0);
    
    // 🎨 方案2: 每个格子内画圆
    vec2 localSt = fract(st);  // 局部坐标 (0~1)
    localSt -= 0.5;  // 中心化
    float circle = smoothstep(0.21, 0.20, length(localSt));
    
    // 🎨 方案3: 砖墙图案
    float row = floor(st.y);
    st.x += step(1.0, mod(row, 2.0)) * 0.5;  // 奇数行偏移
    vec2 brickSt = fract(st);
    float brick = step(0.05, brickSt.x) * step(0.05, brickSt.y);
    
    vec3 color = vec3(brick);
    gl_FragColor = vec4(color, 1.0);
}

🔷 伊斯兰几何图案 ​

glsl
#define PI 3.14159265359
#define TWO_PI 6.28318530718

uniform vec2 u_resolution;

// 🌟 旋转函数
mat2 rotate2d(float angle) {
    return mat2(cos(angle), -sin(angle), sin(angle), cos(angle));
}

void main() {
    vec2 st = gl_FragCoord.xy / u_resolution.xy;
    st = st * 2.0 - 1.0;
    st.x *= u_resolution.x / u_resolution.y;
    
    // 🔷 创建8重对称
    float angle = atan(st.y, st.x);
    float radius = length(st);
    
    // 折叠到1/8扇形
    angle = mod(angle, TWO_PI / 8.0);
    angle = abs(angle - TWO_PI / 16.0);
    
    // 重建坐标
    st = vec2(cos(angle), sin(angle)) * radius;
    
    // 🎨 在折叠空间中绘制简单图案
    float pattern = 0.0;
    pattern += step(0.1, abs(st.x) - 0.3);
    pattern += step(0.1, abs(st.y) - 0.5);
    
    vec3 color = vec3(pattern);
    gl_FragColor = vec4(color, 1.0);
}
