client、offset、scroll区分 🌟 ​

前言 ​

在前端开发中，我们经常需要获取元素的宽高、相对视口的宽高等位置属性。然而，client、offset、scroll等属性常常让人感到困惑。本文将通过详细的解释和示例帮助你理解这些属性。

宽高相关 📏 ​

  1. client：表示可视区域的宽高

    • clientWidth:
      1. 通常指元素的contentWidth + padding。
      2. 对于display: inline的元素（如span），该值为0。
      3. 在怪异盒模型下，等于width - border。
    • clientHeight:
      1. 通常指元素的contentHeight + padding。
      2. 对于display: inline的元素，该值为0。
      3. 在怪异盒模型下，等于height - border。

  2. offset：表示元素的实际占用空间

    • offsetWidth:
      1. 通常指元素的contentWidth + padding + border。
      2. 对于display: inline的元素，该值为实际内容宽度 + padding + border。
      3. 在怪异盒模型下，等于元素的width。
    • offsetHeight:
      1. 通常指元素的contentHeight + padding + border。
      2. 对于display: inline的元素，该值为实际内容高度 + 浏览器默认样式高度 + padding + border。
      3. 在怪异盒模型下，等于元素的height。

  3. scroll：表示元素的滚动区域

    • scrollWidth和scrollHeight通常用于获取文档的实际宽高，包括不可见部分。

  4. 以上属性均为只读属性。

代码示例 ​

html
<style>
  .box1 {
    width: 200px;
    height: 300px;
    border: 2px solid red;
    padding: 20px;
    margin: 100px;
  }
  .box2 {
    box-sizing: border-box;
    width: 200px;
    height: 300px;
    border: 2px solid red;
    padding: 20px;
    margin: 100px;
  }
  .box3 {
    border: 2px solid red;
    padding: 20px;
    margin: 100px;
  }
</style>

<div class="box1"></div>
<div class="box2"></div>
<span class="box3"></span>

<script>
  let box1 = document.querySelector('.box1');
  let box2 = document.querySelector('.box2');
  let box3 = document.querySelector('.box3');

  console.log(box1.offsetWidth); // 244
  console.log(box1.offsetHeight); // 344
  console.log(box1.clientWidth);  // 240
  console.log(box1.clientHeight); // 340

  console.log(box2.offsetWidth); // 200
  console.log(box2.offsetHeight); // 300
  console.log(box2.clientWidth);  // 196
  console.log(box2.clientHeight); // 296

  console.log(box3.offsetWidth); // 44
  console.log(box3.offsetHeight); // 65
  console.log(box3.clientWidth);  // 0
  console.log(box3.clientHeight); // 0
</script>

位置相关 🗺️ ​

  1. client相关

    • clientLeft: 元素左边框宽度
    • clientTop: 元素上边框宽度

  2. offset相关（只读属性）

    • offsetLeft: 元素距离最近的定位包含块的左边框距离
    • offsetTop: 元素距离最近的定位包含块的上边框距离

  3. scroll相关（可读写属性）

    • scrollLeft: 水平方向滚动的距离
    • scrollTop: 垂直方向滚动的距离

事件相关 🎯 ​

  • event.clientX: 目标点距离浏览器可视范围的X轴坐标
  • event.offsetX: 目标点对于触发的源元素的X轴坐标
  • event.pageX: 目标点相对于文档X轴坐标，会计算滚动距离
  • event.screenY: 目标点距离屏幕左侧的X坐标
  • event.getBoundingClientRect().left: 目标元素左边框相对于浏览器可视区域的距离，可能为负值
