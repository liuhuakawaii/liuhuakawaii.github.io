导航栏下划线跟随效果 ​

这是一个常见的导航栏交互效果，当鼠标悬停在导航项上时，下划线会平滑地跟随移动。

实现要点 ​

  • 使用 transition 实现平滑动画
  • 记录最后停留位置，实现鼠标离开时的回归动画
  • 通过 JavaScript 动态计算下划线宽度和位置

效果预览 ​

代码实现 ​

html
<!DOCTYPE html>
<html lang="zh">

<head>
  <meta charset="UTF-8">
  <title>导航栏下划线跟随效果</title>
  <style>
    .nav {
      display: flex;
      position: relative;
      width: 800px;
      margin: 100px auto;
    }

    .nav-item {
      padding: 20px;
      font-size: 18px;
      cursor: pointer;
      text-align: center;
      flex: 1;
    }

    .nav-item:hover {
      color: #1890ff;
    }

    .indicator {
      position: absolute;
      bottom: 0;
      height: 2px;
      background-color: #1890ff;
      transition: all 0.3s ease 0.1s;
    }
  </style>
</head>

<body>
  <div class="nav">
    <div class="nav-item">首页</div>
    <div class="nav-item">产品</div>
    <div class="nav-item">服务</div>
    <div class="nav-item">关于我们</div>
    <div class="indicator"></div>
  </div>

  <script>
    const items = document.querySelectorAll('.nav-item');
    const indicator = document.querySelector('.indicator');
    let lastHoveredItem = items[0]; // 记录最后停留的位置

    // 初始化indicator位置
    function initIndicator() {
      indicator.style.width = `${items[0].offsetWidth}px`;
      indicator.style.left = `${items[0].offsetLeft}px`;
    }

    items.forEach(item => {
      item.addEventListener('mouseenter', () => {
        indicator.style.width = `${item.offsetWidth}px`;
        indicator.style.left = `${item.offsetLeft}px`;
        lastHoveredItem = item;
      });
    });

    // 鼠标离开导航区域时，回到最后停留的位置
    document.querySelector('.nav').addEventListener('mouseleave', () => {
      indicator.style.width = `${lastHoveredItem.offsetWidth}px`;
      indicator.style.left = `${lastHoveredItem.offsetLeft}px`;
    });

    // 初始化
    initIndicator();
  </script>
</body>

</html>

玻璃镜片流光效果 ​

这是一个现代化的卡片设计，结合了玻璃拟态(Glassmorphism)和流光溢彩的动画效果。

实现要点 ​

  • 使用 linear-gradient 创建渐变背景
  • 通过伪元素实现流光动画
  • 利用 opacity 和 transition 控制动画效果

效果预览 ​

关键CSS属性解析 ​

css
/* 玻璃镜片效果 */
.box::before {
  background: linear-gradient(
    110deg,
    rgba(255, 255, 255, 0.9),
    rgba(255, 255, 255, 0) 20%,
    rgba(255, 255, 255, 0) 70%,
    rgba(255, 255, 255, 0.9)
  );
}

/* 流光动画效果 */
.box::after {
  transform: skew(-20deg);  /* 创建倾斜效果 */
  box-shadow: 0 0 10vmin 2.5vmin rgba(255, 255, 255, 1); /* 发光效果 */
}

代码实现 ​

html
<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>玻璃镜片流光效果</title>
  <style>
    /* 容器样式 */
    .container {
      display: flex;
      justify-content: center;
      align-items: center;
      min-height: 100vh;
      background: #1a1a1a;
      /* 深色背景突出效果 */
    }

    /* 卡片基础样式 */
    .box {
      position: relative;
      /* 相对定位，作为伪元素的定位父级 */
      width: 400px;
      height: 200px;
      background: linear-gradient(45deg, #2196f3, #ff4685);
      border-radius: 15px;
      overflow: hidden;
      /* 隐藏溢出部分，使流光效果不超出卡片 */
      cursor: pointer;
    }

    /* 玻璃镜片效果 - 使用::before */
    .box::before {
      content: "";
      position: absolute;
      left: 0;
      top: 0;
      width: 100%;
      height: 100%;
      /* 创建渐变效果，形成玻璃质感 */
      background: linear-gradient(110deg,
          rgba(255, 255, 255, 0.9),
          rgba(255, 255, 255, 0) 20%,
          rgba(255, 255, 255, 0) 70%,
          rgba(255, 255, 255, 0.9));
      opacity: 1;
      transition: opacity 0.65s ease;
      /* 过渡动画 */
    }

    /* 流光效果 - 使用::after */
    .box::after {
      content: "";
      position: absolute;
      background-color: rgba(255, 255, 255, 0.6);
      width: 5vmin;
      top: 0;
      bottom: 0;
      left: -100%;
      /* 初始位置在卡片外 */
      transform: skew(-20deg);
      /* 倾斜效果 */
      /* 发光效果 */
      box-shadow: 0 0 10vmin 2.5vmin rgba(255, 255, 255, 1);
      transition: all 0.25s ease;
    }

    /* 悬停时玻璃效果消失 */
    .box:hover::before {
      opacity: 0;
    }

    /* 悬停时触发流光动画 */
    .box:hover::after {
      left: 400px;
      /* 移动到卡片右侧 */
      transition: left 0.5s ease;
    }
  </style>
</head>

<body>
  <div class="container">
    <div class="box"></div>
  </div>
</body>

</html>

轮盘按钮效果 ​

这是一个创意十足的交互设计，适用于需要展示多个快捷操作的场景。按钮初始呈现为圆形，悬停时展开成扇形菜单。

效果预览 ​

关键技术 ​

  1. 扇形计算
css
/* 扇形区域计算公式：
 * 旋转角度 = -160° + (n-1) * 50°
 * 其中 n 为扇形序号(1,2,3)
 */
.sectorBox:nth-child(1) {
  transform: rotate(-160deg) skewY(40deg);
}
.sectorBox:nth-child(2) {
  transform: rotate(-110deg) skewY(40deg);
}
.sectorBox:nth-child(3) {
  transform: rotate(-60deg) skewY(40deg);
}
  1. 文字方向修正
css
/* 文字需要反向旋转以保持正确方向 */
.sectorBox:nth-child(1) span {
  transform: skewY(-40deg) rotate(160deg);
}

使用注意事项 ​

  • 建议控制菜单项数量在3-5个之间
  • 注意移动端触发区域的大小
  • 可以通过调整 transform-origin 改变展开中心点

文字渐入动画效果 ​

一个优雅的文字动画效果，通过逐字延迟显示创造出流畅的渐入体验。适用于标题、提示文字等需要吸引注意力的场景。

实现原理 ​

  1. 使用 JavaScript 将文本拆分为单个字符
  2. 为每个字符添加延迟动画
  3. 利用 CSS custom properties 控制动画延迟

关键代码解析 ​

javascript
// 文本拆分处理
text.innerHTML = text.textContent
  .split('')
  .map(char => `<span class="letter">
    ${char === ' ' ? '&nbsp;' : char}
  </span>`)
  .join('');

// 设置动画延迟
letters.forEach((letter, index) => {
  letter.style.setProperty('--delay', `${index * 0.05}s`);
});

动画优化建议 ​

  • 使用 cubic-bezier 创建更自然的动画曲线
  • 控制合适的延迟时间间隔（建议 0.05s-0.08s）
  • 考虑添加 will-change 优化性能

性能注意事项 ​

  • 避免在过长文本中使用
  • 考虑使用 requestAnimationFrame 优化动画
  • 添加 transform 硬件加速

效果预览 ​

代码实现 ​

html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>文字渐入动画效果</title>
    <style>
        /* 容器样式 */
        .text-container {
            display: flex;
            justify-content: center;
            align-items: center;
            height: 100vh;
            font-size: 24px;
        }

        /* 文字容器基础样式 */
        .generating-tip {
            position: relative;
            display: inline-block;
        }

        /* 每个字母的基础样式 */
        .letter {
            display: inline-block;
            opacity: 0; /* 初始设置为透明 */
        }

        /* 添加动画效果 */
        .letter.animate {
            /* 使用cubic-bezier创建弹跳效果 */
            animation: moveIn 1s cubic-bezier(0.89, 0.32, 0.84, 1.3) forwards;
            /* animation-delay 将由 JS 通过 --delay 变量设置 */
            animation-delay: var(--delay);
        }

        /* 定义动画关键帧 */
        @keyframes moveIn {
            from {
                opacity: 0;
                transform: translateY(-100%);
            }
            to {
                opacity: 1;
                transform: translateY(0);
            }
        }
    </style>
</head>
<body>
    <div class="text-container">
        <div class="generating-tip">Multiple views, please be patient.</div>
    </div>

    <script>
        // 等待DOM加载完成
        document.addEventListener('DOMContentLoaded', () => {
            const text = document.querySelector('.generating-tip');
            
            // 1. 将文本分割成单个字符并包装在span中
            text.innerHTML = text.textContent
                .split('')
                .map(char => `<span class="letter">${char === ' ' ? '&nbsp;' : char}</span>`)
                .join('');
            
            // 2. 为每个字母设置延迟时间
            const letters = document.querySelectorAll('.letter');
            letters.forEach((letter, index) => {
                // 设置每个字母的延迟时间，每个间隔50ms
                letter.style.setProperty('--delay', `${index * 0.05}s`);
                // 添加动画类
                letter.classList.add('animate');
            });
        });
    </script>
</body>
</html>

中间透明 + 圆角渐变色边框的容器 ​

一个现代化的 UI 容器效果，结合了渐变边框、毛玻璃效果和圆角设计。适用于卡片、弹窗等需要突出视觉层次的界面元素。

实现要点 ​

  • 使用 backdrop-filter 实现毛玻璃效果
  • 通过 mask 属性创建中空边框
  • 利用 CSS 渐变实现边框颜色过渡

效果预览 ​

代码实现 ​

html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>透明渐变边框容器</title>
    <style>
        /* 创建一个深色背景以便更好地展示效果 */
        body {
            background: #1a1a1a;
            display: flex;
            justify-content: center;
            align-items: center;
            min-height: 100vh;
            margin: 0;
        }

        /* 主容器样式 */
        .box {
            width: 500px;
            height: 500px;
            border-radius: 16px;
            /* 添加毛玻璃效果 */
            backdrop-filter: blur(4px);
            padding: 0 min(1.32vw, 30px);
            /* 设置容器背景为半透明渐变 */
            background: linear-gradient(180deg, 
                rgba(0, 0, 0, 0.30) 0%, 
                rgba(0, 0, 0, 0.20) 100%
            );
            position: relative;

            /* 添加一些内容样式 */
            color: white;
            display: flex;
            justify-content: center;
            align-items: center;
            font-size: 24px;
        }

        /* 渐变边框效果 */
        .box::before {
            /* 定义边框相关变量 */
            --border-width: 2px;
            --border-color: linear-gradient(
                10deg, 
                rgb(166, 36, 227) 0%,
                rgb(190, 160, 235) 45%, 
                rgb(190, 160, 235) 55%, 
                rgb(67, 0, 255) 100%
            );
            
            /* 绝对定位并居中 */
            position: absolute;
            content: '';
            left: 50%;
            top: 50%;
            transform: translate(-50%, -50%);
            
            /* 设置尺寸和基础样式 */
            width: 100%;
            height: 100%;
            box-sizing: border-box;
            display: block;
            border-radius: inherit;
            padding: var(--border-width);
            
            /* 设置渐变背景 */
            background: var(--border-color);
            background-clip: padding-box;
            
            /* 关键：使用mask实现中间透明效果 */
            -webkit-mask-image: linear-gradient(red, red), linear-gradient(red, red);
            -webkit-mask-clip: padding-box, content-box;
            mask-composite: exclude;
            
            /* 将伪元素置于内容下方 */
            z-index: -10;
        }
    </style>
</head>
<body>
    <div class="box">
        渐变边框容器
    </div>
</body>
</html>

CSS圆形贴合裁剪效果 🎨 ​

效果介绍 ✨ ​

在现代网页设计中，我们经常需要实现一些独特的视觉效果。今天要介绍的是如何使用CSS实现两个元素之间的圆形贴合裁剪效果，这种效果常用于创建连接的标签、气泡对话框等设计元素。

实现原理 💡 ​

这个效果主要通过以下几个关键技术点实现：

  1. 使用border-radius创建圆形元素
  2. 利用radial-gradient实现圆形裁剪效果
  3. 通过定位和z-index控制元素层级关系

代码实现 🔨 ​

html
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>圆形贴合裁剪效果</title>
  <style>
    body {
      padding: 10px;
    }

    .inversePair {
      border: 1px solid black;
      display: inline-block;
      position: relative;
      height: 100px;
      text-align: center;
      line-height: 100px;
      vertical-align: middle;
    }

    #a {
      width: 100px;
      border-radius: 50px;
      background: grey;
      z-index: 1;
    }

    #b {
      width: 200px;
      padding-left: 30px;
      margin-left: -30px;
      border-left: none;
      background-image: radial-gradient(
        circle closest-corner at -23px 50%, 
        transparent 0, 
        transparent 55px, 
        black 56px, 
        grey 57px
      );
    }
  </style>
</head>
<body>
  <div id="a" class="inversePair">A</div>
  <div id="b" class="inversePair">B</div>
</body>
</html>

关键点解析 🔍 ​

  1. 圆形元素A:

    • 使用border-radius: 50px创建圆形
    • 设置z-index: 1确保显示在上层

  2. 元素B的裁剪效果:

    • 使用radial-gradient创建径向渐变
    • -23px 50%定位渐变的中心点
    • circle closest-corner确定渐变的形状
    • 使用透明色和实色的过渡创建裁剪效果

浏览器兼容性 🌐 ​

为确保更好的兼容性，建议添加各种浏览器前缀：

  • -webkit-radial-gradient
  • -moz-radial-gradient
  • -ms-radial-gradient
  • -o-radial-gradient

使用场景 💼 ​

  1. 标签连接效果
  2. 气泡对话框
  3. 导航菜单设计
  4. 时间轴节点

注意事项 ⚠️ ​

  1. 确保两个元素的高度保持一致
  2. 注意调整间距和裁剪半径以获得最佳效果
  3. 在移动端需要注意响应式适配
