# 前端算法  
## 简介  
1. 数据结构：计算机存储或组织数据的方式  
2. 算法：解决问题的方式  
3. 时间复杂度   
    + 执行当前算法所预估的“花费时间”  
    + 在写代码的过程中，就可以大概知道代码运行的快与慢  
    + 大O表示法《解析数论》：  
        + O表示有很多，O(1) O(n) O(n^2) O(logn)...  
4. 空间复杂度  
    + 执行当前算法需要占用多少内存空间  
    + O(1) O(n) O(n^2)...  
### 常用方法  
1. 栈  
    1. 特点：后进先出，例：进电梯  
    2. 入栈：`arr.push（）`  出栈：`arr.pop（）`  
    ```js  
      //有效的括号 leetCode 20  
      let  isVaild = function(s){
        let stack = []
        for(let i=0;i<s.length;i++){
          const start = s[i]
          if(s[i]=='(' || s[i]=='['||s[i]=='{'){
            stack.push(s[i])  //入栈
          }else{
            const end = s[s.length-1]
            if(end =='('&& start==')' || end =='[' && start==']'|| end =='{' && start=='}'){
              stack.pop()  //匹配出栈
            }else{
              return false
            }
          }
        }  
        return stack.length == 0  //如果不检测长度，则应奇偶分类讨论
      }  
    ```  
    ```js  
        //删除字符串中的所有相邻重复项 leetcode 1047  
        let removeStr = function(s){
          let stack = []
          for(let i=0;i<s.length;i++){
            const end = stack.pop()
            const start = s[i]
            if(end !== start){
              stack.push(start)
              stack.push(end)
            }
          }
          return stack.join('')
        }  
    ```  
    ```js  
        //简化路径 leetCode 71  
        let simplifyPath = function(s){
          let arr = s.split('/')
          let stack = []  
          arr.forEach(item=>{
            if(item && item=='..'){
              stack.pop()
            }else if(item && item!=='.'){
              stack.push(item)
            }
          })
          return '/'+stack.join('/')
        }  
    ```  
2. 队列  
    1. 特点：先进先出  
    2. 入队：`arr.push()`  出队：`arr.shift()`
