基本概念和原理 ​

首先我们了解一些基本概念

体素：体积像素，类似于二维平面的像素点，体素就是一种三维的像素点，体素化是将三维物体转化为体素的过程。你可以把体素理解成一块一块的积木，体素模型就是用积木拼起来的模型

现在我们有这样一个需求：我们需要通过拖拽/上传，把用户提供的模型文件能够渲染成体素模型，这该怎么做呢？

我们可以把整个需求分解成下面几个小步骤：

  1. 实现上传/拖拽文件功能
  2. File --> Model 实现渲染模型功能
  3. 将模型体素化，渲染体素模型
  4. 优化...

注意点：

  1. 'gltf', 'glb', 'dae'的Loader渲染得到的在object.scene中，其他的都是object

     normalizeAndVoxelize(scene) {
       let scaleFactor = 1;
       const boundingBox = new THREE.Box3().setFromObject(scene);
       if (!boundingBox.isEmpty()) {
         const maxDimension = Math.max(
           boundingBox.max.x - boundingBox.min.x,
           boundingBox.max.y - boundingBox.min.y,
           boundingBox.max.z - boundingBox.min.z
         );

         scaleFactor = 2 / maxDimension * 0.95;
         console.log('scaleFactor', scaleFactor);
       } else {
         console.error('Bounding box is empty.');
       }
       return scaleFactor
     }

Three.js体素化原理及实现 ​

接下来就是重点了，体素化原理及实现。 经过不断地面向搜索，面向AI，终于找到了一个体素化的答案。我这里po上原文，并简单讲解一下他的原理及实现。

原文地址：

原理简述：

  • 首先获取到整个模型的Bounding box边界，然后可以根据自定义一个gridsize也就是我们的体素块的大小，把整个Box的每个点都进行遍历，gridsize就是我们的遍历步长，然后判断是否在Bounding box内，如果在，则进行体素化，否则不进行体素化。

  • 重点就是判断是否在Bounding box内。使用THREE.Raycaster从每个候选坐标向指定方向发射一条射线，根据与网格的交点数量判断这些坐标是否确实位于网格内（这个具体就可以看原文，为什么可以这样判断）

  • 使用射线检测的时候，每一个点进行一个方向的射线检测可能不太准确，所以可以增加射线检测的方向，增加到3-6个方向可以提高精确度，但是埋下了性能的隐患

缺点：

  • 射线检测的效率比较低，如果模型体积较大，效率会比较低，所以需要增加射线检测的方向，增加效率，但是埋下了性能的隐患

  • 有些很薄的模型，或者非闭合模型检测可能会有偏差，导致缺体素块，或者多了体素块的问题

      voxelizeMesh(importedScene, voxelName) {
        return new Promise((resolve) => {
          const startTime = (new Date()).getTime()
          console.log('--VOXEL START--');
          const importedMeshes = [];
          importedScene.traverse((child) => {
            if (child instanceof THREE.Mesh) {
              child.material.side = THREE.DoubleSide;
              importedMeshes.push(child);
            }
          });
          let boundingBox = new THREE.Box3().setFromObject(importedScene);
          const xMin = boundingBox.min.x;
          const xMax = boundingBox.max.x;
          const yMin = boundingBox.min.y;
          const yMax = boundingBox.max.y;
          const zMin = boundingBox.min.z;
          const zMax = boundingBox.max.z;
          const meshLength = importedMeshes.length;
          //这里可以自由控制射线检测的数量
          const directions = [
            // new THREE.Vector3(0, 0, 1),
            // new THREE.Vector3(0, 1, 0),
            // new THREE.Vector3(1, 0, 0), 
            new THREE.Vector3(0, 0, -1),
            new THREE.Vector3(0, -1, 0),
            new THREE.Vector3(-1, 0, 0),
          ];

          const pos = new THREE.Vector3();

          for (let i = xMin; i < xMax; i += this.gridsize) {
            for (let j = yMin; j < yMax; j += this.gridsize) {
              for (let k = zMin; k < zMax; k += this.gridsize) {
                pos.set(i, j, k);
                let inside = false;
                for (let meshCnt = 0; meshCnt < meshLength && !inside; meshCnt++) {
                  const mesh = importedMeshes[meshCnt];
                  for (let d = 0; d < directions.length; d++) {
                    if (this.isInsideMesh(pos, directions[d], mesh)) {
                      this.voxels.push({ position: pos.clone() });
                      inside = true;
                      break;
                    }
                  }
                }
              }
            }
          }
          this.voxelsMap = this.voxelsMap.map((item) =>
            item.name === voxelName ? { ...item, voxels: this.voxels, showMesh: importedScene.clone() } : item
          )
          console.log('--VOXEL END--,总耗时：', (new Date()).getTime() - startTime, 'ms');
          console.log('VOXEL SIZE:', this.voxels.length)
          resolve(true)
        })
      }

      isInsideMesh(pos, target, mesh) {
        this.rayCaster.set(pos, target);
        const intersects = this.rayCaster.intersectObject(mesh);
        return intersects.length % 2 === 1;
      }

      recreateVoxels(voxelName) {
        console.log(this.voxelsMap);
        if (this.voxelsMap.length > 1) {
          this.animateVoxels()
          return
        }
        const len = this.voxels.length
        for (let i = 0; i < len; i++) {
          this.dummy.position.copy(this.voxels[i].position);
          this.dummy.updateMatrix();
          this.instancedMesh.setMatrixAt(i, this.dummy.matrix); //将更新后的 this.dummy 对象的变换矩阵设置到 this.instancedMesh 的第 i 个实例
        }
        this.instancedMesh.instanceMatrix.needsUpdate = true;//通知 Three.js 需要更新实例化网格的变换矩阵
        this.voxelsMap = this.voxelsMap.map((item) =>
          item.name === voxelName ? { ...item, instancedMesh: this.instancedMesh.clone() } : item
        )

      }

注意点：

  3. 射线检测的方向，可以自由控制，但是不能太频繁，否则会影响性能

质变优化 ​

从上面的演示图我们可以看见一个很小的模型，检测3方向射线花费时间5s，但是实际得到的体素块并不多，我有尝试过使用八叉树结构进行优化，但是不理想，如果减少射线检测方向，精度会大大降低，生成的模型不尽人意，如果检测6方向射线，时间则会几何倍增，而且模型也不一定准确

经过公司大佬指点，给我提供了一个思路：利用深度图

简述过程如下：

  1. 我们首先获取直接渲染一个16 * 16 * 16 的体素块的正方体

  2. 获取模型6个视角的深度图

  3. 遍历体素块，判断每个体素块的深度图是否在6个视角的深度图中，如果在，则保留，否则删除

是不是有点像高中数学的立体几何？？？？

瞬间豁然开朗！ 这就类似根据三视图来获取立体几何，使用的就是大模型减减小模型，然后通过深度图来判断是否保留体素块，从而实现质变优化！遍历的过程就是一个O(n)的算法

      depthMap2Pos(depthMap, voxelName) {
        // const directions = ["front",'back','right','left','top','bottom']
        const voxelsMappingPosArr = new Array(16 * 16 * 16).fill(1)

        for (let i = 0; i < depthMap.length; i++) {
          const element = depthMap[i].data
          const name = depthMap[i].name
          const distance = 16
          element.map((item, index) => {
            item = Math.ceil(((item / 255 * 2 - 1) - (-1)) / (2 / 16))
            for (let i = 0; i < distance - item; i++) {
              let position
              if (name === 'front') {
                position = { x: index % 16, y: 15 - Math.floor(index / 16) % 16, z: 15 - i }
              } else if (name === 'back') {
                position = { x: 15 - index % 16, y: 15 - Math.floor(index / 16) % 16, z: i };
              } else if (name === 'right') {
                position = { x: i, y: 15 - Math.floor(index / 16) % 16, z: index % 16 };
              } else if (name === 'left') {
                position = { x: 15 - i, y: 15 - Math.floor(index / 16) % 16, z: 15 - index % 16 };
              } else if (name === 'top') {
                position = { x: index % 16, y: i, z: 15 - Math.floor(index / 16) % 16 };
              } else {
                position = { x: index % 16, y: 15 - i, z: Math.floor(index / 16) % 16 };
              }
              const pos = position.x + position.y * 16 + position.z * 16 * 16
              voxelsMappingPosArr[pos] = 0
            }
          })
        }
        this.createNewVoxels(voxelName, voxelsMappingPosArr)
      }

      createNewVoxels(voxelName = 'my-cube', voxelsArr) {
        return new Promise((resolve) => {
          const geometry = new THREE.BoxGeometry(2, 2, 2); // 体素块的边长为2，以适应-1到1的空间
          const material = new THREE.MeshPhongMaterial({ color: 0xffffff, side: THREE.DoubleSide });
          let mycube = this.voxelsMap.find(item => item.name === voxelName)
          mycube.showMesh = new THREE.Mesh(geometry, material);
          mycube.dummy = new THREE.Object3D();
          const min = -1
          const max = 1
          const gridSize = (max - min) / 16; // 每个体素的大小
          const pos = new THREE.Vector3();

          if (voxelsArr) {
            for (let i = 0; i < voxelsArr.length; i++) {
              if (voxelsArr[i] === 1) {
                pos.set((i % 16 + 0.5) * gridSize - 1, (Math.floor(i / 16) % 16 + 0.5) * gridSize - 1, (Math.floor(i / 256) % 16 + 0.5) * gridSize - 1);
                mycube.voxels.push({ position: pos.clone() });
              }

            }
          } else {
            for (let i = 0; i < 16; i += 1) {
              for (let j = 0; j < 16; j += 1) {
                for (let k = 0; k < 16; k += 1) {
                  pos.set((i + 0.5) * gridSize - 1, (j + 0.5) * gridSize - 1, (k + 0.5) * gridSize - 1);
                  mycube.voxels.push({ position: pos.clone() });
                }
              }
            }
          }

          mycube.instancedMesh = new THREE.InstancedMesh(
            new RoundedBoxGeometry(
              gridSize, gridSize, gridSize, // 体素块的大小
              2, this.boxRoundness
            ),
            new THREE.MeshPhongMaterial({ color: 0xffffff, side: THREE.DoubleSide }),
            mycube.voxels.length
          );

          if (voxelsArr) {
            this.voxels = mycube.voxels
            this.instancedMesh = mycube.instancedMesh
            this.dummy = mycube.dummy
            this.scene.add(mycube.instancedMesh);
            this.showMesh.visible = false;
            this.instancedMesh.visible = true;

            this.voxelsMap = this.voxelsMap.map((item) =>
              item.name === voxelName ? { ...item, voxels: this.voxels, showMesh: this.showMesh.clone() } : item
            )

            this.recreateVoxels(voxelName)
          }

          resolve(true);
        });
      }
