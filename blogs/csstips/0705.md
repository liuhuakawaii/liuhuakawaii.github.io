##  flex实现自适应网格布局

1. code
    ```html
          <div className={style.colorPalette}>
              {colorPaletteArr.map((item, index) => {
                 return <div
                  className={`${[0, 1].includes(index) ? style.specialItem : style.nomarlItem} ${style.colorItem}`}
                  style={{
                    backgroundColor: item.color,
                  }}
                  key={index}>
                  {[0, 1].includes(index) ? item.name : ''}
                </div>
              })}
            </div>
    ```

2. css
    ```css
        .colorPalette{
          background-color: #fff;
          border-radius: 8px;
          display: flex;
          flex-wrap: wrap;
        }


        .colorItem{
          display: flex;
          align-items: center;
          justify-content: center;
          position: relative;
          color: #646464;
          font-size: 12px;
          height: 20px;
          border-radius: 4px;
          margin: 5px;
        }
        .nomarlItem{
          flex: 1 0 calc(25% - 10px); /* 每行最多4个元素，占据宽度的25%减去margin */
        }
        .specialItem{
          flex: 1 0 calc(50% - 10px);
        }
    ```
